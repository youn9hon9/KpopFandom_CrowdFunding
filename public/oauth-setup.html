OAuth 설정이 필요합니다

실제 소셜 로그인을 사용하려면 `.env` 파일에 OAuth 자격 증명을 등록해야 합니다. Mock 시뮬레이터는 제거되었습니다.

  1. 프로젝트 루트에서 .env.example을 복사해 .env 생성
  2. REDIRECT_URI_BASE를 접속 URL과 동일하게 설정 (예: http://localhost:3000)
  3. Google Cloud Console / Kakao Developers에서 OAuth 클라이언트 생성
  4. 리다이렉트 URI 등록: /auth/google/callback, /auth/kakao/callback
  5. 서버 재시작 후 다시 로그인
